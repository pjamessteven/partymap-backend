To All Wayback Machine Users,

I’ll get right to it: please support the Internet Archive today. Right now, we have a 2-to-1 Matching Gift Campaign, so you can triple your impact, but time is running out! Most can’t afford to give, but we hope you can. The average donation is $45. If everyone reading this chips in just $5, we can keep the Wayback Machine growing. That’s right, all we need is the price of a paperback book to sustain the leading public archive of the web. When I started this 23 years ago, people called me crazy. Collect web pages? Why? We built the Wayback Machine because online information is fragile. The web needs a memory. No one should be able to change the past. Today, we’ve fixed 11 million broken links in Wikipedia with the Wayback Machine. Meanwhile, we’re dedicated to your privacy so we never track you. We don’t accept ads. But we still need to pay for servers and staff. The Wayback Machine is a real bargain, but we need your help. If you find our site useful, we ask you humbly, please chip in. Thank you.
—Brewster Kahle, Founder, Internet Archive
